# #請假查詢 工作流程技能

## 觸發條件
使用者輸入 `#請假查詢`

## 執行步驟

### Step 1 - 取得今日民國日期
- 取得當前系統時間，轉換為民國紀年（西元年 - 1911）
- 格式：`YYYMMDD`（如民國 115 年 7 月 8 日 → `1150708`）

### Step 2 - 複製查詢工具檔案
- 來源：`D:\北科附工\015-Antigravity工作資料夾\03-Antigravity協助撰寫巨集\差假大批查詢工具.xlsm`
- 目的地：同資料夾，命名格式 `{民國日期}-差假大批查詢工具.xlsm`
  - 例：`1150708-差假大批查詢工具.xlsm`
- 若目的地已存在，詢問使用者是否覆蓋

### Step 3 - 掃描 PDF 資料夾
- 掃描路徑：`D:\北科附工\015-Antigravity工作資料夾\03-Antigravity協助撰寫巨集\002-待作業檔案\`
- 列出所有 `.pdf` 檔案
- 若資料夾為空，通知使用者並終止

### Step 4 - PDF 辨識與解析
對每個 PDF：
- 使用 `pdfplumber` 或 `PyMuPDF (fitz)` 讀取文字內容
- 依據 PDF 類型辨識資料結構：
  - **職場巡輔交通費申請表**：提取姓名、每次巡輔日期（民國格式），預設時段為 `0800~1700`（上午）與 `1300~1700`（下午），依表格欄位判斷
  - **其他類型**：嘗試自動識別姓名欄與日期欄
- 解析民國日期 → 轉換為 `YYYMMDD` 格式
- 每筆資料格式：`[序號, 姓名, 日期YYYMMDD, 開始時間, 結束時間]`

### Step 5 - 填寫查詢介面
- 開啟複製後的 xlsm 檔案（使用 win32com）
- 找到「查詢介面」工作表
- 從第 5 列開始填寫，每筆一列：
  - A: 序號（連續）
  - B: 姓名
  - C: 日期（YYYMMDD，文字格式）
  - D: 開始時間（4碼，如 0800）
  - E: 結束時間（4碼，如 1700）
- 儲存檔案後關閉（確保釋放鎖定）

### Step 6 - 完成通知
- 報告：複製的檔案路徑、已處理 PDF 數量、填入的查詢筆數
- 提示使用者接下來可開啟檔案，匯入差假資料庫後執行查詢

## 注意事項
- 民國日期格式須使用 7 碼（YYY + MM + DD），月份與日期不足兩位須補零
- 時間格式 4 碼（HHMM），如 8 點整 → `0800`
- PDF 辨識後若有無法確定的資料，列出並請使用者確認
- 執行完畢後，`002-待作業檔案` 中的 PDF **不移動、不刪除**（由使用者手動管理）

## 技術棧
- Python + pdfplumber / PyMuPDF
- win32com（操作 Excel xlsm）
- 民國日期轉換：`ROC_YEAR = 西元年 - 1911`
