# Excel 巨集與民國曆日期計算工具箱 (excel-roc-leave-batch-query)

> 一套專門為台灣學校及公務機關設計的 Excel VBA 請假比對巨集工具，完美支援中華民國曆（民國年）與西元日期時間的雙向解析、比對、著色與自動篩選。

---

## 📖 專案簡介 (Overview)

在台灣的學校、公務機關及金融機構中，請假系統所匯出的明細資料往往大量採用中華民國曆（民國紀年，如 `1150210` 或 `115/02/10`）作為日期格式。然而，當行政人員需要針對特定的教職員工名單，批次查詢他們在某段特定時間內（如：`08:00 ~ 09:00`）是否處於請假狀態時，往往面臨著繁瑣的日期比對與計算工作。手動逐一比對不僅耗時費力，更極易因為跨天假單、跨中午休息時間或民國閏年計算而產生疏漏。

本專案旨在提供一個直覺、高效且高度人性化的解決方案。透過精心設計的 Excel 巨集活頁簿，使用者只需將差勤系統匯出的請假明細一鍵匯入，並在查詢介面上輸入「姓名、民國日期、起迄時間」，點擊查詢後，系統便會自動在記憶體中建立高效雜湊字典 (Dictionary)，在 1 秒內精確比對出該時段同仁的差假狀態。

此外，本工具特別注重使用者體驗，設計了雙向著色與自動篩選功能。一旦發現同仁請假，系統不僅會於查詢結果中標記，還會同步在資料庫中著色並自動篩選出該筆原始請假記錄，從根本上解決了行政人員在大量報表間人工對照的痛點，大幅提升了差假核對與管理的效率。

## ✨ 核心功能 (Key Features)

*   **一鍵式匯入請假明細**：支援選擇從學校差假系統中匯出的原始請假檔 (`.xls` 或 `.xlsx`)，自動過濾並排版，將所有請假資料安全無損地載入至隱藏的 `差假資料庫` 工作表。
*   **高速民國曆日期計算 (CNS 7648)**：內建強健的轉換函式，支援 `1150210`、`115/02/10` 或 `115-02-10` 等各種常見格式。具備民國年閏年校驗與天數防呆，直接將民國日期轉換為西元 `Date` 進行精確運算。
*   **精確區間重疊比對演算法**：完美處理「跨天請假」與「相切時段（如前一筆假單到 12:00，下一筆查詢從 12:00 開始）」的複雜行政邏輯，確保比對精準無誤。
*   **雙向著色標註與自動顏色篩選**：
    *   **查詢介面著色**：查詢結果為「有請假」的列會自動標註為淡紅色，並於明細欄詳細顯示其重疊的假別、時間與事由；資料格式錯誤列則標註為淡黃色。
    *   **資料庫同步篩選**：自動將被命中的「差假資料庫」原始請假列同步標註為淡紅色，並**自動啟用 Excel 背景色篩選 (AutoFilter by Cell Color)**，讓使用者在資料庫中一目了然。
*   **無損復原與防呆清除**：
    *   提供一鍵「清除查詢內容」功能，清空所有名單與著色，並自動解除資料庫篩選與標註，**僅還原視圖，不破壞任何已匯入的請假資料**。
    *   欄位採用文字格式，防止 Excel 自動吞掉 `0800` 等時間的前導 `0`。

## 🚀 快速開始 (Quick Start)

### 環境要求
*   Microsoft Excel 2010 或以上版本（需啟用巨集功能）。
*   Windows 10 / 11 作業系統。

### 安裝與運行
1.  下載或複製專案中的巨集活頁簿：[差假大批查詢工具.xlsm](file:///d:/北科附工/015-Antigravity工作資料夾/03-Antigravity協助撰寫巨集/差假大批查詢工具.xlsm)。
2.  開啟檔案，並在 Excel 上方的警告列中點選 **「啟用內容 (Enable Content)」** 以允許巨集執行。
3.  點擊 **「1. 匯入請假明細」** 按鈕，選擇您從差勤系統匯出的請假檔。
4.  在查詢介面的表格中（第 5 行開始）填入欲查詢的名單與時段（日期請輸入 7 碼民國年，如 `1150210`；起迄時間輸入 4 碼，如 `0800`、`1730`）。
5.  點擊 **「2. 執行大批查詢」** 即可產出結果並自動標色篩選。
